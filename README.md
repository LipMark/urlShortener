# url-Shortener
Simple url-shortener service

Prerequisities:
1. Install Golang
2. Install tdm-gcc

How to Run:
1. Clone repository
2. Go get dependencies
3. Change .env to your path(CONFIG_PATH=C:\yourpath...\local.yaml)
4. Go run main.go

Possible operations:
1. Add url with custom alias
2. Get url by alias
3. Delete url by id
